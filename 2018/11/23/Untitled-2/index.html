MySQL的工具命令

2018-11-23 数据库 mysql Comments

1、MySQL表文件：
（1）对于myisam存储引擎来说，不支持事务，支持表锁，每张表有3个文件
.frm：表结构定义文件
.MYD：表空间文件
.MYI：表索引文件

（2）对于InnodeDB存储引擎来说，支持事务，支持行锁，每张表有2个文件
.frm：表结构定义文件
.ibd：表空间文件（存储数据和索引）

2、MySQL工具：
（1）mysql客户端工具：
mysql
mysqldump：备份工具
mysqladmin：管理工具
mysqlcheck：检查工具
mysqlimport：导入工具

（2）mysql非客户端工具：
myisamchk：myisam check，检查工具
myisampack：压缩工具

比如说，用mysqladmin修改密码：
mysqladmin -u USERNAME -h HOSTNAME password “NEW PASSWORD” -p “OLD PASSWORD”
也可以进入命令行修改：
SET PASSWORD FOR ‘USERNAME‘@’HOST’=password(‘NEW_PASSWORD’)

3、MySQL模式：
（1）交互式模式：进入命令行操作MySQL数据库
（2）批处理模式：或者叫脚本模式，在交互式模式下通过source PATH/xx.sql来执行sql脚本。除此之外，还可以在linux命令行下，通过输入重定向来执行脚本，比如#mysql < xx.sql

4、MySQL命令：
（1）客户端命令：不需要发送到服务器执行，而是自身和服务器端交互得到的结果。不需要使用语句结束符

在MySQL命令行模式下的客户端命令：
\d：定义语句结束符
\c：提前终止语句执行
\r：重新连接服务器
\g：无论语句结束符是什么，直接将此语句送至服务端去执行
\G：无论语句结束符是什么，直接将此语句送至服务端去执行，而且是以竖排的形式来显示（纵向显示对于在横排显示不下一张表的时候，是一种非常好的显示手段）
\p：print，显示正在执行的命令
! COMMAND：执行shell命令
\W：大写W表示语句执行之后显示警告信息
\w：小写w表示语句执行之后不显示警告信息
#：支持命令补全功能
备注：
因为要把数据库中的每一个表、每一个字段都载入到内存当中，才能进行命令补全，这会使得连接数据库的时候变得非常慢，因此MySQL默认是禁用命令补全功能的。

禁用补全功能（连接MySQL的时候指定参数）
-A
–no-auto-rehash
–disable-auto-rehash
启用补全功能（在交互式命令行下定义）
#
rehash

在linux命令行模式下的客户端命令
mysqldump
mysqlimport
mysqlcheck
mysqladmin:详细介绍一下mysqladmin
mysqladmin create hellodb #不用连上数据库直接在linux命令行下创建（会读取my.cnf下的数据库的定义）
也可以这样创建：mysqladmin -uroot -p password -h host_name create hellodb
还有一些其他子命令，比如：
mysqladmin drop db_name #删除db
mysqladmin ping -uroot -p -h host_name(ip) ping #测试对方mysql服务器是否在线
mysqladmin processlist #显示正在执行的进程（线程）列表，相当于是连上mysql之后执行的SHOW PROCESSES
mysqladmin status #显示mysql服务器的状态
mysqladmin status –sleep 2 #定义2s显示一次
mysqladmin status –count 2 #定义显示次数为2次
mysqladmin extended-status #显示状态变量（状态变量对于查询mysql服务器状态和排除故障至关重要）
mysqladmin variables #显示服务器变量（跟状态变量区分）
mysqladmin flush-privileges #等同于reload，作用是让mysql重读授权表
mysqladmin flush-status #重置服务器状态变量
mysqladmin flush-threads #重置线程缓存
mysqladmin flush-logs #重置日志滚动
mysqladmin flush-hosts #重置连接的host
mysqladmin refresh #相当于mysqladmin flush-logs和mysqladmin flush-hosts
mysqladmin shutdown #关闭mysql服务器
mysqladmin version #显示当前服务器版本
mysqladmin start-slave #启动复制，启动从服务器复制线程
mysqladmin stop-slave #关闭复制

（2）服务端命令：有语句结束符，默认是分号(可以通过delimiter或者\d来自定义语句结束符)
这里介绍一个最万能的命令：help KEYWORD，比如help CREATE TABLE就是查看CREATE TABLE的用法
我们日常用的mysql命令通常是服务器端命令
