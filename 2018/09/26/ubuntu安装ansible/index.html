ubuntu安装ansible

1、安装add-apt-repository必要套件
apt-get install -y python-software-properties software-properties-common

2、使用ansible官方的PPA套件来源
add-apt-repository -y ppa:ansible/ansible

3、升级apt-get
apt-get update

4、安装ansible
apt-get install -y ansible
